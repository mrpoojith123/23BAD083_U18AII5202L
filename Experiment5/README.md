## Objective
Perform statistical analysis and exploratory visualization on student performance data to identify patterns, correlations, and performance distributions across different departments and subjects.

## Dataset
**File:** `5.student_performance.csv`

The dataset contains 52 student academic records with the following attributes:
| Column | Description |
|--------|-------------|
| Student_ID | Unique identifier for each student |
| Department | Department (IT, AI&DS, CSE) |
| Semester | Current semester (5, 6, 7) |
| Subject | Subject name (Statistics, DAA, ML, Python, DBMS) |
| Internal_Test1 | Marks in Internal Test 1 |
| Internal_Test2 | Marks in Internal Test 2 |
| Assignment_Marks | Assignment marks |
| Attendance_Percentage | Student attendance percentage |
| Final_Grade | Final grade awarded (A, B, C, D) |

## Libraries Used
- Base R and statistical packages

## Analysis Performed
Statistical exploration and descriptive analysis including:
- Summary statistics and distributions
- Performance comparisons across departments
- Subject-wise performance metrics
- Grade distribution analysis
- Correlation between test scores and assignments

## Key Insights
- Analyze academic performance patterns
- Identify subject difficulty levels
- Correlate attendance with final grades
- Department-wise performance comparison
